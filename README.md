# Catálogo V4 — Standalone (Next.js)

Projeto **100% Next.js (App Router)** + **Tailwind** no padrão **preto + dourado**.

## Rotas
- `/` → Home / Landing
- `/catalogo` → Lista de produtos (placeholder)
- `/produto/[id]` → Tela de produto / click-review (placeholder)

## Rodar local
```bash
npm install
npm run dev
```
